# Z-Money-Finance-Tracker
 ZMoney adalah aplikasi manajer keuangan pribadi yang dirancang untuk membantu pengguna mengelola, melacak, dan menganalisis pemasukan serta pengeluaran mereka dengan mudah. Dengan antarmuka yang bersih dan intuitif, ZMoney mempermudah pencatatan transaksi harian,dan pemahaman terhadap kebiasaan finansial Anda untuk mencapai tujuan keuangan.
